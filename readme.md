# Sobre o projeto

Este é um portfólio simples desenvolvido com HTML, CSS e JavaScript.

O objetivo do projeto é apresentar uma estrutura básica de página pessoal, contendo uma seção "Sobre mim" e uma listagem de projetos renderizada dinamicamente a partir de um array em JavaScript.

Além disso, o layout foi construído utilizando Flexbox e possui responsividade básica para diferentes tamanhos de tela.
